# 🎓 Student Affairs & Services Management System (SASMS)

## 📌 Project Overview
The **Student Affairs & Services Management System (SASMS)** is a web-based platform designed to manage the complete student lifecycle, from online admission to daily services and communication.

The system focuses on student administrative operations only and does **NOT include academic results or grading system**.

---

## 🚀 Key Features

### 📝 1. Online Admission System
- Student application submission
- Document upload
- Application status tracking (Pending / Accepted / Rejected)

### 👨‍🎓 2. Student Information Management
- Student profiles
- Class and department assignment
- Personal data management

### ⏰ 3. Attendance & Departure System
- Daily attendance tracking
- Entry/exit records
- Monitoring student presence

### 💰 4. Fees Management
- Fee payments tracking
- Payment history
- Outstanding balances

### 📢 5. Complaints & Suggestions System
- Submit complaints
- Send suggestions
- Admin response tracking

### 🎭 6. Student Activities Management
- Manage extracurricular activities
- Participation tracking

### 💬 7. Messaging & Notifications System
- Internal communication system
- System notifications for students and admins

### 📊 8. Dashboards
- Student dashboard
- Admin dashboard
- Overview statistics

---

## 🛠️ Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML5 / CSS3
- JavaScript
- Bootstrap
